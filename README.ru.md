# Calc123

Calc123 — компактный WordPress-плагин для создания калькуляторов на основе формул.

Текущая версия плагина: **1.3**.

Плагин позволяет создать калькулятор в админке WordPress, задать входные переменные и формулу, а затем вывести форму на странице или в записи через шорткод. Формулы обрабатываются через tokenizer, обратную польскую запись и стековый вычислитель без использования `eval()`.

## Состояние проекта

Версия 1.3 — текущая очищенная публичная база. Старый экспериментальный слой лицензирования намеренно не включён в этот репозиторий.

Плагин работоспособен в существующем объёме. Дальнейший рефакторинг и расширение функциональности планируются только при наличии практической необходимости, результатов тестирования или требований совместимости.

## Возможности

- Несколько калькуляторов в административной части WordPress.
- Создание, редактирование, дублирование и удаление калькуляторов.
- Вывод через шорткод: `[calc123 id="N"]`.
- Операторы формул: `+`, `-`, `*`, `/`, `^`, скобки и сравнения.
- Поддерживаемые функции: `IF(cond,a,b)`, `MAX(a,b)`, `MIN(a,b)` и `ROUND(a,decimals)`.
- Переменные типов number и select с возможностью скрыть поле на фронтенде.
- Варианты select в формате `Метка:значение`.
- Изменение порядка переменных и ширины полей: `1/1`, `1/2`, `1/3`, `1/4`.
- Необязательный текст или обозначение валюты до или после результата.
- Необязательный собственный ID и CSS-класс контейнера формы.
- Простая математическая капча.
- AJAX-расчёт без перезагрузки страницы.
- Клиентская и серверная валидация.
- Без `eval()`, Composer-, npm-зависимостей и внешних сервисов.
- Без собственных таблиц базы данных: калькуляторы хранятся через WordPress Options API.

## Проверенная конфигурация

Текущая публичная версия вручную проверена на:

- WordPress 7.0;
- PHP 8.4;
- MySQL 5.7.

Другие версии WordPress, PHP и базы данных могут работать, но проверены не во всех сочетаниях. Перед использованием на рабочем сайте плагин следует протестировать на копии проекта.

## Установка

### Установочный ZIP из GitHub Release

Используйте файл `calc123.zip`, приложенный к GitHub Release, когда релиз опубликован. Автоматические архивы GitHub **Source code** содержат весь репозиторий и не предназначены для прямой загрузки через установщик плагинов WordPress.

### Ручная установка из репозитория

1. Скопируйте папку `calc123` в `/wp-content/plugins/`.
2. Активируйте **Calc123** в разделе **Плагины → Установленные плагины**.
3. Откройте **Calc123** в основном меню админки WordPress.
4. Создайте калькулятор, задайте переменные и формулу.
5. Вставьте полученный шорткод в запись или страницу.

## Базовое использование

Создайте калькулятор в административной части и вставьте его шорткод в контент:

```text
[calc123 id="1"]
```

Пример формулы:

```text
IF(distance > 100, 1000 + weight * 50, 500 + weight * 30)
```

Пример вариантов для select-поля:

```text
Стандарт:10000,Премиум:20000
```

Коды переменных лучше задавать латинскими буквами, цифрами и символом подчёркивания, чтобы их можно было однозначно использовать в формулах.

## Встроенная справка

Файл `calc123/about.html` содержит описание и примеры, которые открываются в административном окне плагина. Файл `.htaccess` закрывает прямой HTTP-доступ к этой справке на Apache-серверах, а WordPress загружает её через PHP.
