# Calc123

Calc123 — небольшой WordPress-плагин для создания калькуляторов на основе формул.

Плагин позволяет создать калькулятор в админке WordPress, задать переменные, прописать формулу в привычной математической записи и вывести форму на странице через шорткод.

Проект намеренно сделан компактным: без визуального конструктора, без зависимости от внешних сервисов и без использования `eval()` для выполнения формул.

## Возможности

- Несколько калькуляторов, управляемых из админки WordPress.
- Вывод через шорткод: `[calc123 id="N"]`.
- Операторы формул: `+`, `-`, `*`, `/`, `^`, скобки и сравнения.
- Поддерживаемые функции: `IF(cond,a,b)`, `MAX(a,b)`, `MIN(a,b)`, `ROUND(a,decimals)`.
- Типы переменных: число, выпадающий список и скрытое поле.
- Необязательный текст до/после результата, например обозначение валюты.
- Необязательная простая математическая капча перед расчётом.
- AJAX-расчёт без перезагрузки страницы.
- Клиентская и серверная валидация.
- Разбор формулы через tokenizer → RPN → stack evaluator, без `eval()`.

## Установка

Загрузите папку `calc123` в каталог:

```text
/wp-content/plugins/calc123/
```

Затем активируйте **Calc123** в разделе **Плагины → Установленные плагины**.

В папке плагина должны быть файлы:

```text
calc123.php
calc123-frontend.js
calc123-frontend.css
about.html
.htaccess
```

Файл `about.html` используется в админке плагина для окна с описанием и примерами. Файл `.htaccess` закрывает прямой HTTP-доступ к `about.html` на Apache-серверах. Сам плагин при этом читает файл из PHP.

## Использование

Создайте калькулятор в разделе **Calc123**, добавьте переменные и формулу, после чего вставьте шорткод на страницу или в запись:

```text
[calc123 id="1"]
```

Пример формулы:

```text
IF(distance > 100, 1000 + weight * 50, 500 + weight * 30)
```

Пример формата вариантов для select-поля:

```text
Standard:10000,Premium:20000
```
